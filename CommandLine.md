## Command line arguments: ##

### Syntax: ktanc2e.exe -(switch) ###

-d (folder)	- source directory/folder

-mp3		- process MP3 files

-epub		- process epub files

-s			- include subdirectories

-r			- rename files

-?			- show this message


_Example:_ **ktanc2e.exe -d c:\MyDocs -epub -mp3 -s -r**
